# 首页

星域世界创立于 **2018 年 8 月 14 日**，是一个 Minecraft 国际版服务器。服务器以长期运行为目标，通常情况下会持续开放 Mod 服务器，并保持稳定的运维节奏。欢迎你邀请朋友一同加入。

在不违反《玩家守则》的前提下，你可以在服务器内自由游玩，按照自己的方式探索、建造与交流。我们不会限制你“应该怎么玩”，也不要求你以固定节奏参与游戏。你可以在玩家交流群内讨论其他游戏或话题；即使暂时不游玩服务器，也欢迎正常交流。

任何事物都不可能尽善尽美。若你在游玩过程中遇到问题，或对服务器有改进建议，欢迎随时反馈。我们会认真对待每一条合理的意见，并持续调整与完善。

星域世界是一个**长期运行、持续维护的服务器项目**。周目会不断更替，玩法会不断变化，但服务器本身长期存在。

我们相信，Minecraft 的乐趣不只来自“保存结果”，也来自一次次新的开始、不同方向的尝试，以及每一周目独特的体验。

在这里，你可以选择长期投入，也可以选择阶段性回归；可以专注建造与发展，也可以只在你感兴趣的时候参与。

我们为**长期稳定运行**而奋斗，为一个可以放心离开、也能随时回来的服务器而奋斗。

我们坚持：

- 每个周目都会引入新的玩法方向，保持活力
- 运维以稳定、可持续为第一目标
- 不做快餐服，长期开服，不跑路
